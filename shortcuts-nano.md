# Shortcuts : nano

- <kbd>Ctrl</kbd><kbd>A</kbd> Move cursor to start of line
- <kbd>Ctrl</kbd><kbd>E</kbd> Move cursor to end of line
- <kbd>Ctrl</kbd><kbd>Y</kbd> Move cursor one page up
- <kbd>Ctrl</kbd><kbd>V</kbd> Move cursor one page down
- <kbd>Ctrl</kbd><kbd>K</kbd> Cut line
- <kbd>Ctrl</kbd><kbd>U</kbd> Paste line
- <kbd>Alt</kbd><kbd>U</kbd> Undo
- <kbd>Alt</kbd><kbd>E</kbd> Redo
- <kbd>Alt</kbd><kbd>Backspace</kbd> Delete word to the left
- <kbd>Ctrl</kbd><kbd>Delete</kbd> Delete word to the right
- <kbd>Alt</kbd><kbd>Delete</kbd> Delete line
